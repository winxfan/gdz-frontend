# Архитектура проекта

- App Router структура
- Типизация всех пропсов
- MUI темизация
- API routes для бэкенда

# Конвенции кода

- Именование: PascalCase для компонентов, camelCase для функций
- Файловая структура: app/(routes)/page.tsx, components/ui/
- Стили: предпочтительно sx пропс над styled-components
